============================================
    BACKUPS AUTOMATICOS - INSTRUCCIONES
============================================

TODOS LOS DESARROLLADORES DEBEN HACER ESTO:

⚠️ SI LA VENTANA SE CIERRA MUY RAPIDO:
   - Primero ejecutar: diagnostico_backup.bat
   - Como administrador
   - Te dira que esta fallando

1. ACTUALIZAR PROYECTO:
   - git pull
   - composer install

2. CONFIGURAR BACKUPS:
   - Buscar archivo: configurar_backup_automatico.bat
   - Clic derecho -> "Ejecutar como administrador"
   - Decir "Si" cuando pregunte
   - IMPORTANTE: presionar ENTER cuando te pida
   - Esperar hasta ver simbolos verdes ✓

3. VERIFICAR:
   - Doble clic en: verificar_estado_backup.bat
   - Debe mostrar simbolos verdes ✓

============================================

SI HAY PROBLEMAS:
- Ventana se cierra rapido: usar diagnostico_backup.bat
- Error "acceso denegado": ejecutar como administrador
- Error "PHP no encontrado": instalar PHP
- Error "no es Laravel": ejecutar desde carpeta correcta
- Error "backup no funciona": hacer composer install
- Otros errores: tomar captura y enviar al grupo

QUE HACE:
- Crea backups automaticos cada 2 minutos
- Funciona aunque cierres Visual Studio Code
- No verás ventanas ni notificaciones
- Se puede cambiar frecuencia desde la web

IMPORTANTE: SOLO SE HACE UNA VEZ, NO CADA ACTUALIZACION
